# DAA Shortest Path Visualizer

This project demonstrates the implementation of **Dijkstra's Algorithm** and **Breadth First Search (BFS)** using Python.  
It calculates the shortest path between cities and visualizes the route on an interactive map.

## Project Description

The program takes a **start city** and a **destination city** as input.  
It then uses **Dijkstra's Algorithm** to find the shortest distance and path between them.

The project also:
- Displays BFS traversal order
- Shows the distance table from the source city
- Measures execution time of the algorithm
- Generates a visual map of the route using Folium

## Algorithms Used

### 1. Dijkstra Algorithm
Used to find the **shortest path between two cities** in a weighted graph.

### 2. Breadth First Search (BFS)
Used to traverse all cities from the starting node.
